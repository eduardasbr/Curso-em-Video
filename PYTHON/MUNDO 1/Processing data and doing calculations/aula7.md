
## ➕ Operadores Aritméticos

| Operador | Nome                | Exemplo |
|----------|---------------------|--------|
| `+`      | Adição              | `2 + 2` |
| `-`      | Subtração           | `5 - 3` |
| `*`      | Multiplicação       | `2 * 3` |
| `/`      | Divisão             | `10 / 2` |
| `**`     | Potência            | `2 ** 3` |
| `//`     | Divisão inteira     | `10 // 3` |
| `%`      | Resto da divisão    | `10 % 3` |
