
# 📘 Aula 7 — Tipos Primitivos e Operadores

---

## 🔢 Tipos Primitivos

### 📌 Inteiros (`int`)
- Exemplos: `0`, `-1`, `1024`

### 📌 Reais (`float`)
- Exemplos: `4.5`, `0.076`, `-15.223`, `7.0`

### 📌 Booleanos (`bool`)
- Valores: `True`, `False`

### 📌 Strings (`str`)
- Exemplos: `"Olá"`, `"7.5"`, `""`
